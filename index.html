Get Started

What is AIGP.md?

AI Global Preferences (AIGP.md) is a simple file format that allows you to define your preferences for AI tools. Store it as ~/AIGP.md in your home directory, and supported AI clients will automatically use these preferences across all your interactions.

Think of it as your personal AI configuration file - a place to specify your coding style, communication preferences, and tool choices that follow you between different AI platforms.

Why AIGP.md?

Consistency

Set your preferences once, use them everywhere. No more re-configuring the same settings across different AI tools.

Portability

Your preferences move with you between AI platforms. Switch tools without losing your setup.

Control

Define how AI tools interact with you. Specify coding standards, communication style, and project preferences.

How to use

1

Create the file

Create AIGP.md in your home directory (~/AIGP.md).

2

Add preferences

Write your preferences in simple Markdown format. Include coding style, project preferences, and interaction guidelines.

3

Use with AI tools

Use AI clients that support AIGP.md. They'll automatically read your preferences and apply them to all interactions.
